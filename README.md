## Running the server locally

After downloading the repository
Install the gems

```
bundle
```

Install tailwind css

```
rails tailwindcss:install
```

Create the database

```
rails db:create
```

Migrate the database

```
rails db:migrate
```

Seed the database

```
rails db:seed
```

Start server:

```
rails s
```

Visit localhost:3000 to see the app.
